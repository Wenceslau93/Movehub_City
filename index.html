Desafio Movehub City¶

Definindo o Problema de Negócio¶

O dataset Movehub City, se refere a uma mtrica onde mostra o ranking das melhores cidades para se viver.

Para avaliar se uma cidade boa para se viver, necessrio levar alguns pontos em considerao, como por exemplo:

  • Se a cidade possui uma boa localidade;
  • Populao tem uma boa qualidade de vida
  • Populao com salrios maiores que a mdia;
  • Se a chances de crescimento em termo de valorizao do local;
  • Segurana do local;
  • Existe pouca taxa de crime;
  • Entre outras avaliaes.

Pesquisando sobre o dataset para entender mais da regra de negcio, existe uma varivel que se chama Movehub Rating, que a combinao de todas as pontuaes para realizar a classificao geral da cidade.

Portando Movehub Rating ser a varivel dependente, enquanto as demais seram variveis independente.

Para realizar a previso deste modelo, ser necessrio considerar os fatores existentes dentro do conjunto de dados, pos eles contribuem para a obteno do resultado do ranking.

O modelo tem como objetivo realizar previso dos dados, para classificar o ranking da cidade, no qual se refere se boa para se viver.

Ser utilizado o Matplotlib e Seaborn para criao dos grficos.

Utilizando Scikit-Learn, ser utilizada para implementao de um algoritmo de regresso para a modelagem dos dados.

def mostrar_correlacao(data, hurdle = 0.0):
    R = np.corrcoef(data, rowvar = 0)
    R[np.where(np.abs(R) < hurdle)] = 0.0
    heatmap = plt.pcolor(R, cmap = mpl.cm.coolwarm, alpha = 0.8)
    heatmap.axes.set_frame_on(False)
    heatmap.axes.set_yticks(np.arange(R.shape[0]) + 0.5, minor = False)
    heatmap.axes.set_xticks(np.arange(R.shape[1]) + 0.5, minor = False)
